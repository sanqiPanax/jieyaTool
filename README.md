# 解压小工具

一个轻量级的 Windows 解压辅助工具，解决「压缩包后缀名被修改后无法直接解压」的烦恼。


![演示](演示.mp4)
## 适用场景

- 从网上下载的压缩包后缀名被改成了 `.abc`、`.xyz` 等非标准格式
- 需要批量解压一个文件夹下的所有压缩包
- 解压带密码的压缩包
- 分卷压缩包（如 `.part1.rar`、`.7z.001` 等）

## 前置条件

电脑上需要安装以下至少一种解压软件：

- **WinRAR**（推荐）
- **7-Zip**（推荐）
- 或其他支持命令行解压的工具

> 如果没有安装，可选择「自定义」手动指定解压工具路径。

## 快速开始

1. 双击运行 `解压小工具.exe`
2. 拖入文件或点击「选择文件」「选择文件夹」
3. 确认目标后缀和密码（可选）
4. 按 `Enter` 或点击「开始解压」

## 功能详解

### 🔧 解压工具选择

程序启动后自动检测已安装的解压工具，下拉框可选：

- **WinRAR**：从注册表自动查找
- **7-Zip**：从注册表自动查找
- **自定义**：未检测到时，选择此项会出现路径输入框

### 🔑 密码记忆

- 解压成功后自动记录使用的密码
- 下次打开自动选中上次使用的密码
- 下拉列表按使用频率排序
- 密码数据保存在同目录下的 `config.json` 中

### 📦 分卷压缩支持

- WinRAR 新风格：`archive.part1.rar`、`archive.part2.rar`
- WinRAR 旧风格：`archive.rar`、`archive.r00`、`archive.r01`
- 7-Zip 分卷：`archive.7z.001`、`archive.7z.002`

### 🔄 密码容错

对于没有加密的压缩包，即使密码框中有内容也能自动无密码解压，无需手动清空密码。

## 常见问题

**Q：提示"解压工具未找到"怎么办？**

A：在下拉框中选择「自定义」，然后点击「浏览」选择电脑上的 `WinRAR.exe` 或 `7z.exe`。

**Q：解压失败怎么排查？**

A：检查以下几点：
1. 目标后缀名是否设置正确
2. 压缩包是否损坏
3. 分卷压缩包是否齐全
4. 解压工具路径是否正确

**Q：如何清除密码历史？**

A：删除同目录下的 `config.json` 文件即可。
